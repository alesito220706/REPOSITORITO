📋 Soluciones Rápidas
1️⃣ Usa BD Local (H2) para Desarrollo

Ejecuta la aplicación con el perfil local para usar H2 en memoria:

mvnw spring-boot:run -Dspring-boot.run.arguments="--spring.profiles.active=local"
2️⃣ Verifica Conexión a Render

Si deseas usar PostgreSQL en Render.com, verifica:

  • El servidor está activo en render.com
  • Las credenciales en application.properties son correctas
  • Tienes conexión a internet estable
3️⃣ Reinstala Base de Datos Render

Si la BD está corrompida, bórrala y crea una nueva en Render
